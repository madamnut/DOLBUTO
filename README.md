# DOLBUTO
Vulkan / C++

## Development

- Open this folder in Visual Studio as a CMake project.
- Select the `DOLBUTO` target.
- Build manually from Visual Studio.

## Controls

- Mouse: rotate camera
- W/S: move forward/backward horizontally
- A/D: move left/right horizontally
- Space: move up
- Shift: move down
- Left click: capture mouse
- Middle click: release mouse
- F3: toggle debug text
- F11: toggle fullscreen
- Esc: close window

## Debug Text

- Top left: `FPS: 0000 [000.000MS]`
- Under FPS: player position, yaw/pitch, and 4-way facing
- Top right: `DOLBUTO 0.0.0.0`
- Under version: CPU and GPU hardware names without labels
- Under hardware: Vulkan API and driver versions
- Under driver: current resolution
